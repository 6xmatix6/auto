2015 Mercedes-Brabus G500 4x4²

Installation (Add-On)
1. Drop the "2015MercedesBrabusG5004x4" folder to: mods\update\x64\dlcpacks
2. Extract "dlclist.xml" or edit it with OpenIV from mods\update\update.rpf\common\data\
3. Use notepad or OpenIV to open/edit it, and add the new line:
		<Item>dlcpacks:\2015MercedesBrabusG5004x4\</Item>
4. Save / replace it in OpenIV
5. Use Menyoo or Simple Trainer and spawn it by name: Brabus500

Installation (Replace):
1. Drop the YTD & YFT files to your latest Patchday folder
Example: GTA 5\mods\update\x64\dlcpacks\patchdayXXng\dlc.rpf\x64\levels\gta5\vehicles.rpf
